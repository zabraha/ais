# AI Powered Knowledge Base Search User Interface
A simple UI prototype React + TypeScript knowledge article search interface with filtering, sorting.

## Tech Stack
React 19 + TypeScript
CSS Grid/Flexbox 

## Project Structure

src/  
├── types.ts              # Article, Category types  
├── data.ts              # Mock articles dataset  
├── hooks/  
│   └── useDebouncedValue.ts  
├── components/  
│   ├── SearchBar.tsx  
│   ├── FiltersPanel.tsx  
│   ├── ResultCard.tsx  
│   ├── ResultsList.tsx  
│   └── ArticleDetailModal.tsx  
├── App.tsx              # Main app  
└── index.css           # css styles  

# Quick Start

## Clone & install
npm install

## Run dev server
npm start

## Build for production
npm run build
